# Cross-Cloud Interconnect (Interconexión Multi-Nube)

Establece conexiones físicas directas y privadas desde Google Cloud hacia otras nubes públicas como AWS o Azure.

- **Caso de uso:** Arquitecturas multi‑nube críticas que transfieren grandes cantidades de datos entre proveedores de nube pública de forma privada y sin pasar por Internet.
- **SLA:** Hasta 99.99 % de disponibilidad (según la topología y los acuerdos de nivel de servicio de los proveedores involucrados).
- **Requisitos técnicos clave:** Las cargas de trabajo deben estar en nubes soportadas (AWS, Azure, OCI) y se necesita Cloud Router para el intercambio dinámico de rutas BGP entre las VPCs.
